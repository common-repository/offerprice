== Description ==

Sell any product or service? Collect price offers! Add "Offer price" link to your page and display the offer price form to your prospect customers.

== Installation ==

Upload the OfferPrice plugin to your blog, activate it,
then [log in to OfferPriceWidget](http://www.offerpricewidget.com/users/sign_up),
generate WordPress shortcode, and paste the shortcode to your page or post.

This section describes how to install the plugin and get it working.
